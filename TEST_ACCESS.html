Troubleshooting

If you encounter "Outdated Optimize Dep" errors:

  1. Stop the development server
  2. Run: rm -rf node_modules/.vite
  3. Run: rm -rf node_modules && npm install
  4. Restart: npm run dev

The application should now be accessible at http://localhost:5174
